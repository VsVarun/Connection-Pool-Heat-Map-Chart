Animated Heat Map Chart for Connection Pool.
The Chart will let the user to visualize the connections pool usage status.
This is simplied by directly accepting a Connection Pool Status object.

In an Actual Scenario
   When threads awaiting connections, there will not be any Idle or Open connections.
   
Total Number of connections visualized is computed as follows

    SuperTotal = Total(Active + Idle + Open) + Waiting

Active  - Connections acquired/held by Thread or connections in Use or Connections in Transaction
Idle    - Connections Opened but Idle
Open    - Free/Available Connection Slot
Total   - Total allocated Connections in the Connection Pool
Waiting - Threads Awaiting to accquire a Connection from the Connection Pool

:::Javascript:::

// Sample data
var connectionPoolStatus = {
    Name: 'com.zaxxer.hikari:type=Pool (HikariCP)',
    Total: 500,
    Active: 370,
    Idle: 50,
    Waiting: 30
};

// To invoke
// Syntax
connectionPoolHeatMap(heatMapElementName, data, shufle, renderType, animateFlag, boxSize);

// Aguments
HeatMapElementName   = '#div id'
ConnectionPoolStatus = { Name: 'Sample', Total: 500, Active: 370, Idle: 50, Waiting: 10 }
shuffle              = [ true or false ]
renderType           = [ TABLE or SVG or SnapSVG or CANVAS ]
animateFlag          = [ true or false ]
boxSize              = [ 1 - 10 ]

// Example 
connectionPoolHeatMap('#ConnectionPoolHeatMap', connectionPoolStatus, true, 'SnapSVG', true, 4);
